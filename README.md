🚀 SpendWise – AI Powered Expense Tracker

📌 Overview

SpendWise is a full-stack personal finance management application that helps users track expenses, manage subscriptions, analyze spending behavior, and gain financial insights.

It is built using modern web technologies with a focus on performance, usability, and real-world financial tracking.

✨ Features

💰 Expense Management
Add, edit, delete expenses
Category-based expense tracking
Payment method tracking
Notes and date-based entries

Interactive dashboard with charts

🚀 Getting Started
1. Clone Repository
git clone https://github.com/your-username/spendwise.git
cd spendwise
2. Install Dependencies
npm install
3. Setup Environment Variables

Create .env.local file:

NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
4. Run Development Server
npm run dev
5. Build for Production
npm run build
